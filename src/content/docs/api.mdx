# API

Базовый префикс — `/api/v1`. Все запросы требуют заголовок
`Authorization: Bearer <TOKEN>`. Тело загрузки — `multipart/form-data`,
остальные запросы и ответы — `application/json`.

<Callout type="info" title="Ориентир">
  Сигнатуры ниже соответствуют контракту doc-service. Точные поля для вашей
  версии всегда сверяйте со Swagger/OpenAPI вашего инстанса.
</Callout>

## Задания (jobs)
