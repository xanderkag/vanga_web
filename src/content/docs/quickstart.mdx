# Быстрый старт

Минимальный happy-path интеграции — от файла до данных за четыре шага.

1. **Отправляете файл** в `POST /api/v1/jobs` → получаете `job_id`.
2. **Сервис обрабатывает** документ асинхронно (OCR → классификация →
   извлечение → валидация).
3. **Узнаёте о готовности** — вебхуком на ваш `webhook_url` или опросом
   `GET /api/v1/jobs/{job_id}`.
4. **Забираете JSON** с извлечёнными полями.

<Callout type="info" title="Авторизация">
  Все запросы к <code>/api/v1/*</code> требуют заголовок
  <code>Authorization: Bearer &lt;TOKEN&gt;</code>. Токен выдаётся при
  подключении к пилоту.
</Callout>

## Шаг 1. Загрузка документа

  samples={[
    {

  -H "Authorization: Bearer $TOKEN" \\
  -F "file=@invoice.pdf" \\
  -F "webhook_url=https://your.app/hooks/docparser"`,
    },
    {
      label: "Node",
      language: "javascript",
      code: `const form = new FormData();
form.append("file", await fileFromPath("invoice.pdf"));
form.append("webhook_url", "https://your.app/hooks/docparser");

const res = await fetch("https://api.docparser.example/api/v1/jobs", {
  method: "POST",
  headers: { Authorization: \`Bearer \${TOKEN}\` },
  body: form,
});
const { job_id } = await res.json();`,
    },
    {
      label: "Python",
      language: "python",
      code: `import requests

res = requests.post(
    "https://api.docparser.example/api/v1/jobs",
    headers={"Authorization": f"Bearer {TOKEN}"},
    files={"file": open("invoice.pdf", "rb")},
    data={"webhook_url": "https://your.app/hooks/docparser"},
)
job_id = res.json()["job_id"]`,
    },
  ]}
/>

Ответ:

```json
{ "job_id": "5e9c…", "status": "pending" }
```

## Шаг 2. Дождаться результата

Рекомендуемый путь — **вебхук**: как только job достигает терминального
статуса (`done`, `failed` или `needs_review`), сервис сам пришлёт POST на ваш
`webhook_url`. Подробнее — в разделе [Вебхуки](/docs/webhooks).

Если вебхук недоступен — **опрос статуса**:

<CodeTabs
  samples={[
    {
      label: "curl",
      language: "bash",
      code: `curl https://api.docparser.example/api/v1/jobs/$JOB_ID \\
  -H "Authorization: Bearer $TOKEN"`,
    },
    {
      label: "Python",
      language: "python",
      code: `import time, requests

while True:
    job = requests.get(
        f"https://api.docparser.example/api/v1/jobs/{job_id}",
        headers={"Authorization": f"Bearer {TOKEN}"},
    ).json()
    if job["status"] in ("done", "failed", "needs_review"):
        break
    time.sleep(2)`,
    },
  ]}
/>
