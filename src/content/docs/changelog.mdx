# История версий

## Портал

### 1.0 — 2026-06
- Запуск продуктового портала: лендинг, документация, интерактивное демо,
  форма заявки на пилот.
- Документация для интеграторов: API, типы документов, вебхуки, ошибки.
- Тёмная/светлая тема, адаптив, базовый SEO.

## API (doc-service)

### v1 — текущий контракт
- `POST /api/v1/jobs` — загрузка документа и постановка в очередь.
- `GET /api/v1/jobs/{id}` — статус и результат; `GET /api/v1/jobs` — список.
- Вебхуки с подписью HMAC-SHA256, ретраями и идемпотентностью по `job_id`.
- 15 встроенных типов документов; 6 базовых — с типизированной схемой.
- Доменная валидация: ИНН/КПП, НДС, госномера, сходимость сумм.
- OCR-каскад: pdf-text → tesseract → vision-llm → yandex.

<Callout type="info" title="Совместимость">
  Добавление необязательных полей в payload вебхука обратно-совместимо и
  остаётся на v1. Breaking-изменения поднимают версию с параллельной доставкой.
</Callout>
